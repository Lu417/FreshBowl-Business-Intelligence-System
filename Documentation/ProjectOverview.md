# FreshBowl Business Intelligence System

## Project Background

FreshBowl is a fictional healthy fast-food chain operating across South Australia.

The purpose of this project is to simulate a real-world Business Intelligence solution using:

- SQLite
- SQL
- Python
- Power BI
- GitHub

The system is designed to analyse sales performance, customer behaviour, store performance, inventory status, and staff productivity.

---

## Database Structure

The database contains eight core tables:

- Customers
- Products
- Stores
- Staff
- Orders
- OrderItems
- Inventory
- Suppliers
